# Predicting Survival on the Titanic (a Kaggle competition)
In this project, we explored the Titanic dataset and trained a simple decision tree model that ranked 1st among 89 teams on the private leaderboard. The model achieved 0.82035 and 0.83665 accuracy on the public and private leaderboard respectively. Please find in [this notebook](https://github.com/huiyiQ/titanic/blob/main/titanic.ipynb) a walkthrough of our work.
